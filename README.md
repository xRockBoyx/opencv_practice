
# 初始設定

## 安裝conda

[Conda Download](https://www.anaconda.com/products/distribution)

## 建立虛擬conda環境
```
conda create -n python3.7 python=3.7
```

## 進入虛擬環境
```
conda activate python3.7
```

## 安裝套件指令
```
pip install [packagename]==[version]
```

## 離開虛擬環境
```
conda deactivate
```
